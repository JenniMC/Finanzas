# Finanzas Personales 2023 - 2025
Es una pequeña pueba de las librerias **Pandas, Mathplotlib, Numpy, Seaborn**

Durante la prueba se hace tratamiento de datos con la eliminacion de celdas vacias e inncesarias, asi como tambien el cambio de celdas NULL a celdas con 0.

Se crea una nueva tabla con dos columnas de la original con la finalidad de crear un grafico que muestre el flujo.
